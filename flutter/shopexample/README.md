# shopexample

创建命令：
`flutter create --platform web shopexample`

这是一个flutter演示项目

# 目录结构
```shell
- lib
  - api         # 接口层
  - assets      # 存放资源
  - components  # 公共组件
  - contants    # 常量文件
  - viewmodels  # 类型文件(types)
  - pages       # 存放页面
  - routes      # 路由配置
  - stores      # 迁居状态组件
  - utils       # 工具类
  - main.dart   # 程序入口
```

# 基础路由和组件
路由有两个：
- `/`：主页
- `/login`：登录页
