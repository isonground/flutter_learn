# 创建web项目
命令:
`flutter create --platforms web create_project`

在终端运行:
```shell
cd create_project
flutter run
```
然后选择对应浏览器

# 项目结构
```shell
  .dart_tool              # dart工具生成的文件和缓存
  .idea                   # Intellij IDEA配置文件
  build                   # 项目构建目录,包含编译生成文件
* lib                     # 项目主要源代码目录
    main.dart           # 应用程序入口
  test                    # 测试代码目录
  web                     # web平台特定的配置和资源文件
  .gitignore              # Git版本控制忽略文件配置
  metadata                # 项目标识文件,包含项目信息,如名称、版本、依赖等,(自动生成)
  analysis_options.yaml   # 配置静态代码分析工具
  create_project.iml      # 用于存储模块(Module)的特定设置
  pubspec.lock            # 依赖锁定文件
* pubspec.yaml            # 项目依赖和配置文件
```
其中*标注了主要需要关心的文件

# flutter启动文件说明
在`main.dart`文件中,通过`runApp`函数启动应用程序。
`runApp`函数的参数`const MyApp()`是一个`Widget`对象,用于构建应用程序的UI。
